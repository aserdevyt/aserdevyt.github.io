quick menu

aserdev-os's quick menu can be accessed by pressing super + I.

and it is very helpfull if you get used to it.

quick menu options:

  1. settings: open's settings
  2. run command: gives a menu to run a command
  3. websearch: allows you to quickly search the web
  4. notifications: view recent notifications using swaync
  5. install: install applications or updates
  6. clipboard: access clipboard history
  7. emoji picker: pick and insert emojis
  8. screen capture: capture screenshots or record your screen
  9. kill window: close the currently focused window
  10. discord: open Discord application
  11. systeminfo: view system information
  12. power: access power options such as shutdown, restart, and logout
  13. about: opens my github repository
