Aserdev-OS

Aserdev-OS is a minimal, fast, and customizable arch-based distribution.
Designed for people like me who like a fast and minimal system powered by Arch Linux.

⚠️ work in progress — don't use if you are not sure ⚠️

This os is inspired by omarchy but it isn't dev focused — it's focused for people who just want a cool Hyprland setup that isn't fancy or complicated. (aka: me)

Installation

Script (recommended)

Make a fresh install of Arch Linux and a minimal setup with multilib and testing enabled. Create a normal user with sudo and make sure pipewire is installed. After reboot into your minimal Arch install run:

sudo pacman -Sy curl sh -c "$(curl -fsSL https://raw.githubusercontent.com/aserdevyt/aserdev-os/main/install.sh)"
